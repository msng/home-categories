== Description ==

Filters the categories of posts shown on your front page.

= How to Use =

1. After Installation, activate the plugin. Don't worry, it does nothing when activated for the first time.
2. Go to **Settings** > **Home Categories** to setup.
* Select the working mode from **Show selected categories only** or **Hide selected categories**
* Check the target categories.
* Press **Save Changes** .

== Installation ==

1. Upload `home-categories` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
